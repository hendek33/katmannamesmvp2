========================================
ELEVENLABS SOUND EFFECTS SETTINGS
========================================

GENERATION PARAMETERS:
- Use detailed, specific prompts
- Include instrument/material types when relevant
- Specify duration in the prompt if needed
- Describe the emotion/feeling desired
- Include environmental context (reverb, echo, etc.)

PROMPT OPTIMIZATION TIPS:
- Be specific about materials (metal, wood, glass)
- Describe the space (small room, large hall, outdoor)
- Include timing details (quick, slow, rhythmic)
- Specify pitch (high, low, medium)
- Add texture details (crisp, muffled, echoing)

CATEGORIES FOR ORGANIZATION:
- UI/Interface sounds
- Game state changes
- Player actions
- Ambient/Background
- Dramatic moments
- Team-specific
- Victory/Defeat
- Transitions

OUTPUT SPECIFICATIONS:
- Format: MP3 or WAV
- Sample Rate: 44.1kHz or 48kHz
- Bit Depth: 16-bit or 24-bit
- Normalize to -3dB
- Trim silence at start/end

IMPLEMENTATION NOTES:
- Pre-generate static effects
- Cache all generated sounds
- Use variations for repeated sounds
- Consider layering multiple effects
- Test on different devices/speakers
